# Personal Portfolio Website

This is my personal portfolio website, It's designed to showcase my skills, projects, and professional background in a clean, modern, and interactive way.

## Features

- **Fully Responsive:** The website is built to be fully responsive, providing an optimal viewing experience across all devices.
- **Interactive UI:** Smooth animations and transitions are used throughout the site to create an engaging user experience.
- **Dynamic Content:** The site features sections for an introduction, skills, featured projects, a resume, and a contact form.
- **Project Showcase:** A dedicated section to display key projects with descriptions, tech stacks, and links.
- **Resume Section:** Includes a timeline for education and experience, plus a direct download link for the full resume.


## Technologies Used

- **HTML5:** For the website's structure and semantic markup.
- **CSS3:** For advanced styling, layout (including Flexbox and Grid), and animations.
- **JavaScript (ES6):** Used for DOM manipulation, event handling, and implementing interactive features like the mobile menu and scroll animations.
- **Font Awesome:** For scalable vector icons used throughout the site.
