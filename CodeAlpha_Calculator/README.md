# Simple Calculator

A sleek and functional calculator built with HTML, CSS, and JavaScript. This project provides a user-friendly interface for performing basic arithmetic operations.

## Features

- **Modern & Responsive Design:** The calculator is designed to be intuitive and visually appealing, adapting to different screen sizes.
- **Complete Arithmetic Operations:** Supports addition, subtraction, multiplication, and division.
- **Keyboard Support:** Use your keyboard to input numbers and perform calculations for a faster experience.
- **Error Handling:** Gracefully handles invalid expressions to prevent crashes.
- **Clear & Delete:** Easily clear the entire display or delete the last character.

## Technologies Used

- **HTML:** For the structure of the calculator.
- **CSS:** For styling and creating a modern look and feel.
- **JavaScript:** For the calculator's logic and interactivity.
