# Responsive Image Gallery

This project is a beautiful and responsive image gallery created with HTML, CSS, and vanilla JavaScript. It allows users to view a collection of images, filter them by category, and see them in a full-screen lightbox view.

## Features

- **Filterable Grid:** Easily filter images by categories like 'Nature', 'Tech', and 'Travel'.
- **Interactive Lightbox:** Click on any image to open it in a full-screen, high-resolution lightbox.
- **Easy Navigation:** Navigate through the images in the lightbox using next/previous buttons or keyboard arrow keys.
- **Responsive Design:** The gallery is fully responsive and looks great on all devices, from mobile phones to desktops.
- **Hover Effects:** Smooth hover effects on images provide a better user experience.


## Technologies Used

- **HTML5:** For the basic structure and content of the gallery.
- **CSS3:** For styling, layout (including CSS Grid), and animations.
- **JavaScript (ES6):** To handle the filtering logic, lightbox functionality, and event handling.
- **Font Awesome:** For icons used in the UI.
